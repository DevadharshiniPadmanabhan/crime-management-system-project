# Crime Management System

## Project Overview
The **Crime Management System** is a web-based application developed to manage crime records efficiently in a digital environment. The system helps in maintaining FIR records, tracking case progress, assigning investigating officers, storing evidence, and generating reports. It replaces the traditional manual process with a computerized system to improve efficiency, accuracy, and accessibility of crime-related information.

This project provides a centralized platform where users can register FIRs, view case details, update case status, and maintain complete case history. The system also includes a dashboard for displaying total FIRs, completed cases, pending cases, and current month case analysis.


## Features
- FIR Registration  
- View and Manage Cases  
- Case Status Update (Completed / Incomplete)  
- Investigating Officer Assignment  
- Evidence Management  
- Case History Tracking  
- Reports Dashboard  
- Current Month Case Analysis

## Project Modules

### 1. FIR Registration Module
Allows users to register new FIRs by entering necessary details such as complainant information, crime type, date, and location.

### 2. Case Management Module
Enables users to view all registered cases and update case status as completed or incomplete.

### 3. Officer Assignment Module
Allows assignment of investigating officers for each case to ensure proper responsibility and tracking.

### 4. Evidence Management Module
Stores and manages evidence related to crime cases for maintaining proper case history.

### 5. Reports Module
Displays summary reports such as total FIRs, completed cases, and incomplete cases.

### 6. Monthly Analysis Module
Shows the number of crime cases registered in the current month for analysis purposes.


## How the System Works
1. User registers an FIR through the application.  
2. FIR details are stored in the database.  
3. Users can view all registered cases.  
4. Case status can be updated as completed or incomplete.  
5. Investigating officers can be assigned to cases.  
6. Evidence can be added and maintained.  
7. Reports and case analysis are generated automatically.
